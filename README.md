# TAFP-GraphRAG — JISA Submission Package

This private repository contains only the files prepared for submission to the Journal of Information Security and Applications (JISA).

## Structure

- `01_MANUSCRIPT/` — locked V11 manuscript and review PDF
- `02_FIGURES/` — separate Figure 1–7 files
- `03_SUPPLEMENTARY/` — canonical supplementary package and mapping
- `04_SUBMISSION_METADATA/` — highlights, biography, declarations, and author photograph
- `05_QA_AND_AUDITS/` — scientific lock and final QA evidence
- `06_MANIFESTS/` — authoritative inventories and SHA-256 records
- `07_HANDOFF/` — portable MASTER HANDOFF for continuation in a new chat

Scientific content is locked at V11. This repository does not contain the legacy experiment workspace, seed configuration files, temporary outputs, or retired manuscript versions.
